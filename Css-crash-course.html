This page is your complete guide to CSS — the language that styles the web. By the end, you'll understand its purpose, syntax, and how to use it to design beautiful pages.

What is CSS?

CSS (Cascading Style Sheets) is the language used to style and format HTML content.

  • Cascading – Styles can be inherited and overridden in a specific order.
  • Style – CSS defines how elements look (colors, fonts, sizes, layouts).
  • Sheets – Styles are stored in rulesheets, separate from HTML.

CSS Syntax

CSS uses a simple pattern: selector { property: value; }

p {
  color: red;
  font-size: 16px;
}

This styles all <p> tags with red text and 16px font size.

CSS Selectors

  • Element Selector – p {} targets all paragraphs.
  • Class Selector – .classname {} targets elements with a specific class.
  • ID Selector – #idname {} targets an element with a specific ID.
  • Group Selector – h1, h2 {} targets multiple elements.

CSS Examples

Example of a styled heading:

h1 {
  color: blue;
  text-align: center;
  border-bottom: 2px solid black;
}

Summary

CSS works with HTML to control the look and feel of your website. Combined with JavaScript, it can make your pages both beautiful and interactive.
