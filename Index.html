Welcome to HTML Learning!

This page is your complete guide to HTML — the language that powers the web. By the end, you'll understand its purpose, structure, and how to write it yourself.

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It tells your web browser what to display and how to structure it.

  • HyperText – Links that connect web pages together.
  • Markup – Special symbols (tags) that describe how content should appear.
  • Language – A set of rules that browsers understand.

Basic HTML Structure

Every HTML document follows a structure like this:

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

HTML Elements & Tags

HTML uses tags inside angle brackets, such as <p> for paragraphs. Most tags come in pairs:

  • <h1>...</h1> – Main heading
  • <p>...</p> – Paragraph
  • <img src="image.jpg" alt="Description"> – Image
  • <a href="https://example.com">Link</a> – Link

Hyperlinks

Hyperlinks connect one page to another. Example:

<a href="https://www.example.com">Visit Example</a>

This creates a clickable link.

Summary

HTML is the foundation of the web. Combined with CSS (for styling) and JavaScript (for interactivity), it allows you to build beautiful, functional websites.
